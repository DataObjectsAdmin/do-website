Platform Capabilities

Everything your application needs.

A complete platform — from database design to live deployment — built on infrastructure that scales to millions of records and billions of files.

🗃️
Visual Database Designer
Build tables, fields, and relationships with a point-and-click interface. Text, numbers, dates, selects, checkboxes, files — all field types included.
🎨
Drag-and-Drop Layout Builder
Design detail views and list views with rows, columns, and tabbed layouts. No HTML required. What you design is what users see.
📁
Native File & Media Storage
Images, galleries, PDFs, videos, and any file type stored as first-class data in CloudFlare R2 or your own S3/Azure bucket. Video thumbnails generated automatically.
🌍
Global Data Architecture
Store your data in US East, US West, Europe, UK, Australia, or Asia. CloudFlare CDN accelerates delivery worldwide with no latency compromises.
🔐
Enterprise-Grade Security
2FA via email and SMS. IP and country-based access control. Per-table user roles and granular permission groups. Built in from day one.
📱
Mobile & Browser Ready
Every application adapts automatically to phones, tablets, and desktops. Any modern browser. No app to install. No plugin. No compatibility matrix.
🗄️
Isolated Per-App Databases
Every app runs in its own dedicated SQL database — never sharing tables with other customers. Faster queries, cleaner exports, real data isolation.
🔌
Bring Your Own Storage
Connect your own AWS S3, Azure Blob, or CloudFlare R2 bucket. Your files stay in your infrastructure. No vendor lock-in, ever.
🤖
AI Search — Coming Soon
Upload thousands of PDFs, spreadsheets, and photos. Search file content and images with natural language. Ask your data and get real answers.
The Object Model

Everything is an Object.

A DataObjects app is just a collection of Objects — tables, fields, layouts, users, and rich content blocks. Drop a Map Object onto a layout. Add a Gallery Object to a record. Wire a QR Object to your inventory. The same building blocks compose into anything.

📝
Text
Short text, long text, rich text. Any length, any field.
🖼️
Gallery
Photo galleries with thumbnails, full-screen viewer, drag-and-drop upload.
📎
File
PDFs, spreadsheets, documents — viewable inline, downloadable, searchable.
🗺️
Map
Google Maps embedded directly in records. Auto-geocodes from address fields.
📅
Date & Time
Date pickers, time pickers, datetime combos with timezone support.
☑️
Choice
Dropdowns, radio buttons, checkboxes powered by value lists.
📱
QR Code Soon
Generate and scan QR codes natively inside any layout.
⛅
Weather Soon
Live weather data tied to any location field.
📊
Chart Soon
Bar, line, and pie charts that visualize your database records.
📆
Calendar Soon
Date-based record views built directly into your layouts.
⭐
Rating
Star ratings, score sliders, thumbs up/down for any record.
🔗
Portal
Show related records from another table inline — like a list inside a record.
🛜
NFC Soon
Connect NFC tags directly into your database.

And many more on the roadmap — Signature, Barcode, Audio, Video, SMS, Webhook, Custom API.
